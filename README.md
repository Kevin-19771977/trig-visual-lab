# 三角函數視覺化教學系統

這是一套以 Streamlit 製作的互動式三角函數教學平台，教學順序從初學到進階：

1. 從直角三角形認識三角比
2. 三角比與相似三角形
3. 廣義角與同界角
4. 單位圓與六個三角函數
5. 特殊角三角函數值
6. 三角函數基本圖形
7. 三角函數圖形的伸縮與平移
8. 平方關係、商數關係、餘角關係
9. 學生探索任務與教師引導

## 安裝套件

```bash
pip install -r requirements.txt
```

## 執行方式

```bash
streamlit run app.py
```

## 檔案說明

- `app.py`：主程式
- `requirements.txt`：需要安裝的套件
- `README.md`：使用說明
